# Prodigy InfoTech Web Development Internship 🚀

This repository contains a collection of interactive front-end web development projects built with HTML, CSS, and vanilla JavaScript. It was created as part of the Prodigy InfoTech Web Development Internship tasks (`PRODIGY_WD_02`).

## 📂 Included Projects

### 1. Responsive Navigation Menu (`/navigation-menu`)
A dynamic, interactive, and responsive navigation bar that adapts seamlessly to different screen sizes. Features smooth hover effects and mobile-friendly hamburger menu toggling.

### 2. Interactive Stopwatch (`/stopwatch`)
A fully functional time-tracking web application. It includes start, stop, and reset functionalities with a clean, easy-to-read user interface built for precision.

### 3. Tic-Tac-Toe Game (`/tic-tac-toe`)
The classic grid game built with JavaScript logic. It features turn-based gameplay, automatic win-detection, draw detection, and a reset button to play multiple rounds.

### 4. Personal Portfolio (`/portfolio`)
A personalized developer portfolio website. Designed to be visually appealing and professional, it showcases skills, projects, and contact information with a modern aesthetic and responsive layout.

### 5. Real-Time Weather App (`/Weather App`)
A dynamic weather dashboard with a premium glassmorphism UI. Features include:
- Real-time weather data fetching via Open-Meteo API (No API key required).
- User geolocation detection for localized weather forecasts.
- Search functionality to check the weather of any global city.
- Dynamic background gradients that adapt based on current weather conditions (e.g., sunny, rainy, stormy, etc.).

## 🛠️ Technologies Used
- **HTML5**: Semantic structure and content.
- **CSS3**: Advanced styling, flexbox/grid layouts, responsive media queries, glassmorphism, and animations.
- **JavaScript (ES6+)**: DOM manipulation, event handling, game logic, and asynchronous API data fetching.

## 🚀 How to Run
To view or run any of these projects:
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Phaneendra1228/PRODIGY_WD_02.git
   ```
2. Navigate into any of the project directories (e.g., `cd "Weather App"`).
3. Simply open the `index.html` file in your preferred web browser. No local servers or build steps are required!
